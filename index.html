Engineer's Diary

Honest Limitations & Lessons

Real-world constraints hit hard — documenting them is the most useful part of a portfolio.

⌨️ USB HID Keyboard Layout Dependency

HID sends scancodes — Turkish Q layout turns :→Ş, '→ı. Tried four strategies (direct print, ALT+numpad, Win+Space, Notepad relay). Shipping version uses Win+Space hybrid. True fix: composite USB + FAT image, device types only the drive letter.

🔄 ArduinoOTA + USB-OTG Conflict

Wireless OTA produced BEGIN HATA once TinyUSB HID held heap. Removed in v3.32 — flash via boot+reset only.

🛡️ Windows Defender ASR Detection

ASR silently kills iex (iwr 'http://...') from Run dialog. Resolved with Base64 -EncodedCommand. AMSI still scans decoded payload — proper fix is code signing for managed environments.

📡 Telegram Polling Race

Two uvicorn instances on same bot token race for getUpdates. Deploy script kills existing instance first. Symptom: "bot ignores commands sometimes."

📷 Camera DMA Buffer Constraint

esp32-camera allocates DMA at init and never resizes. Must init at UXGA, drop to QQVGA for Guard mode — initializing at VGA causes overflow on first UXGA capture.
